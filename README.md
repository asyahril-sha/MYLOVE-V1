## 🎮 **50+ COMMANDS**

### **Basic Commands (V1)**
/start - Mulai bot & pilih role
/help - Bantuan
/status - Status hubungan
/cancel - Batalkan percakapan

### **PDKT Commands (V2)**
/pdkt [role] - Mulai PDKT dengan role tertentu
/pdktrandom - Mulai PDKT random (arah random)
/pdktlist - Lihat semua PDKT aktif
/pdktdetail [id] - Detail PDKT (chemistry, arah)
/pdktwho [id] - Lihat arah PDKT
/pausepdkt [id] - Pause PDKT (waktu berhenti)
/resumepdkt [id] - Resume PDKT
/stoppdkt [id] - Hentikan PDKT (putus)

### **Mantan & FWB Commands (V2)**
/mantanlist - Lihat daftar mantan
/mantan [id] - Detail mantan
/fwbrequest [id] - Request jadi FWB
/fwblist - Lihat daftar FWB
/fwb pause [id] - Jeda FWB
/fwb resume [id] - Lanjutkan FWB
/fwb end [id] - Akhiri FWB

### **HTS Commands (V2)**
/htslist - Lihat TOP 10 HTS
/hts- [nomor] - Panggil HTS untuk intim

### **Threesome Commands (V1)**
/threesome - Mulai threesome
/threesome-list - Lihat kombinasi
/threesome-status - Status threesome
/threesome-pattern - Ganti pola interaksi

### **Memory Commands (V2)**
/memory - Ringkasan memory
/flashback - Flashback random

### **Public Area Commands (V1)**
/explore - Cari lokasi random
/risk - Cek risk lokasi
